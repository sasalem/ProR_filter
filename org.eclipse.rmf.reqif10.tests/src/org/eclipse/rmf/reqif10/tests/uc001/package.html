UC001 "Interchange"

To check whether ReqIF tools can import/export that complies to the ReqIF standard. i.e. All reqif files are valid with respect to the reqif schema and the reqif constraints.

The following test data is defined:

TC1000
Simple Content
TC1100
Unformated System Attributes & Spec Hierarchy
TC1200
Formated Content
TC1300
Spec Relations
TC1400
Internal Table
TC1500
Access Restrictions
TC1600
External Content (Multifile)
TC1700
Spec Relations (Multifile)
