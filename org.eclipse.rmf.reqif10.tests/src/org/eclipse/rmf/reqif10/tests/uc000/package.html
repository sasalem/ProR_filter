UC000 "Data model with XML serialization"

To check whether RMF can properly make use of ReqIF as data model and to check of the XML serialization supports all aspects supported by the metamodel

The following test goals are addressed

  • All non abstract metaclasses are instantiated at least once
  • Each feature is set least once
  • Each feature is unset at least once
  • Tool extensions that are known are serialized and deserialized
  • Tool extensions that are unknown are serialized and deserialized
Note: it is not required that the serialized artifacts are valid with respect to the XML schema. E.g. it is allowed that mandatory elements are not serialized or that the root element is not the reqif element.
