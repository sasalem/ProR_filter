Essentials are an extension to ProR that contain useful functionality, but are not part the Eclipse distribution. Features include: XHTML support (formatted text) and Suspect Links (set a flag if source or target of a link changes).

Essentials are free and can be installed via Help | Install new Software... Select ProR Essentials from the "Work with:" dropwdown and follow the standard installation procedure.

More information can be found at the ProR Essentials site.

The ProR Platform for Requirements Engineering and Formal Modeling

ProR is a tool for requirements engineering that supports the ReqIF 1.0.1 Standard natively. It is Eclipse-based and is built for extensibility.

  • More Information? Please visit the ProR page at Eclipse for the latest information. ProR is part of the Requirements Modeling Framework (RMF).
  • Found a Bug? ProR and RMF are still in Incubation and currently under active development. Please use with care. You can report bugs, or ask questions in the RMF Forum.
  • Need more functionality? Consider installing ProR Essentials (see box to the right).
  • Regular news and information? Consider subscribing to the ProR Newsletter.
  • Do you want to get started? Just close this view and start creating requirements.
